# 直通车投放实战指南

## 一、直通车基础

### 计费方式
- **CPC（按点击付费）**：主流方式，每次点击扣费
- **实际扣费 = （下一名出价 × 下一名质量分）/ 你的质量分 + 0.01元**
- **质量分越高，实际扣费越低**

### 质量分三大因素
| 因素 | 权重 | 优化方法 |
|------|------|----------|
| 创意质量 | 40% | 主图点击率>3%，标题包含核心关键词 |
| 相关性 | 30% | 关键词与宝贝类目/属性/标题匹配 |
| 买家体验 | 30% | DSR评分、转化率、收藏加购率 |

### 展现位置
- **关键词推广**：搜索结果页第1、7、13...位（左侧标注"广告"）
- **定向推广**：猜你喜欢、购物车下方、我的淘宝
- **智能推广**：系统自动匹配人群和出价

## 二、新手起步（月预算3000元以下）

### 第1周：测图测款
```
预算分配：每天100元
投放方式：标准推广 > 关键词推广
选词策略：5-10个精准长尾词
出价策略：市场均价的80%，逐步上调
目标：找出点击率>3%的主图
```

### 第2周：优化关键词
```
操作：
1. 删除7天无展现的词
2. 降低点击率<1%的词出价30%
3. 提升点击率>3%的词出价20%
4. 添加相关长尾词10个
5. 否定不相关词（精确否定+短语否定）
```

### 第3-4周：提升ROI
```
操作：
1. 聚焦转化率>2%的关键词
2. 开启定向推广（智能投放）
3. 设置分时折扣（低谷期50%，高峰期100%）
4. 设置地域投放（排除偏远地区）
5. 开启搜索人群溢价（老客户+收藏加购人群溢价30%）
```

## 四、ROI优化公式

### 核心公式
```
ROI = 客单价 × 转化率 / CPC

示例：
客单价 = 128元
转化率 = 3%
CPC = 1.2元
ROI = 128 × 0.03 / 1.2 = 3.2

意味着每花1元广告费，带来3.2元销售额
```

### ROI提升路径
1. **降低CPC** → 提高质量分 → 优化主图点击率
2. **提升转化率** → 优化详情页+评价+价格
3. **提升客单价** → 关联销售+套餐+满减

### ROI健康标准
| 阶段 | 可接受ROI | 良好ROI | 优秀ROI |
|------|----------|---------|---------|
| 测图测款期 | 1:1 | 1:1.5 | 1:2+ |
| 起量期 | 1:1.5 | 1:2 | 1:3+ |
| 稳定期 | 1:2 | 1:3 | 1:5+ |

## 五、常见问题

### Q: 直通车开了没展现？
- 出价太低 → 逐步加价0.1元测试
- 关键词太冷门 → 换搜索量更大的词
- 质量分太低 → 优化主图和标题
- 时间段不对 → 切换到流量高峰时段

### Q: 有展现没点击？
- 主图不吸引人 → 参考竞品高点击图，A/B测试
- 排名太靠后 → 提高出价或优化质量分
- 价格无竞争力 → 调整定价策略

### Q: 有点击没转化？
- 详情页问题 → 优化卖点+评价+买家秀
- 流量不精准 → 否定不相关词，收窄人群
- 价格偏离市场 → 参考同类竞品定价
- 客服响应慢 → 设置自动回复+快捷话术

### Q: ROI一直上不去？
- 先看转化率：低于1%是详情页问题
- 再看点击单价：高于行业均值是词太贵
- 最后看客单价：低于80元很难盈利
