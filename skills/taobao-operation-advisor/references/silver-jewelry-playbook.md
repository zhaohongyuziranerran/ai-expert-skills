## 三、标题优化

### 银饰标题公式
```
925纯银 + 品类词 + 工艺词 + 场景词 + 人群词

示例：
✅ 925纯银手镯女 古法匠心推光 生日礼物送妈妈 素圈光面
✅ 925银项链女 ins风锁骨链 送女友生日礼物 低调轻奢
✅ 925纯银耳钉 防过敏气质耳环 职场通勤日常百搭
```

### 标题禁忌
- ❌ 含"纯银"不含"925"→ 容易被投诉材质不符
- ❌ 堆砌无关热词 → 影响搜索权重
- ❌ 用"足银999"实际925 → 虚假宣传
- ❌ 抄竞品标题一模一样 → 同质化无优势

## 五、定价策略

### 成本结构
```
原材料成本：银价×重量（约4-8元/克）
加工费：15-50元/件
包装费：5-15元/件
平台佣金：5%（天猫2%+3%技术服务费）
推广费：10-20%（直通车+引力魔方）
→ 总成本 = 原材料 + 加工 + 包装 + 佣金 + 推广
→ 建议售价 = 总成本 × 2.5-4倍
```

### 定价参考
| 产品类型 | 成本范围 | 建议售价 | 毛利率 |
|---------|---------|---------|--------|
| 引流款银耳钉 | 8-15元 | 39-69元 | 55-70% |
| 利润款银手镯 | 25-60元 | 99-299元 | 60-70% |
| 形象款银套装 | 80-150元 | 399-899元 | 65-75% |
