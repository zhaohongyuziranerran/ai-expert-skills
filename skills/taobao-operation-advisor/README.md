# AI淘宝运营军师 🛒

> 15年淘宝天猫运营经验，操盘过年GMV超8000万的银饰珠宝类目店铺

## 这是什么？

AI淘宝运营军师是一个专业电商运营技能包，专为淘宝/天猫卖家打造，特别强化了银饰珠宝类目的运营策略。

- 🔍 **5维诊断** — 标题关键词/点击率/转化率/销量权重/DSR评分全面诊断
- 📊 **竞品分析** — 4步法拆解竞品，找到差异化机会
- 💰 **直通车优化** — 从测图测款到ROI稳定3+的完整路径
- 💎 **银饰专属** — 标题公式/详情页模板/定价策略/客服话术全覆盖
- 📅 **7天行动计划** — 诊断→方案→执行→复盘的闭环
