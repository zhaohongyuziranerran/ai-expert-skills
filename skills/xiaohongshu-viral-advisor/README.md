# 小红书爆款军师 📕

> 8年全平台自媒体运营经验，累计打造爆款内容1000+条，全网播放超20亿次

## 这是什么？

小红书爆款军师是一个AI Agent技能包，帮你从0到1打造小红书爆款内容。不管你是新手还是有一定粉丝的博主，它都能提供：

- 🎯 **赛道验证** — 帮你找到"你擅长+你热爱+市场需要"的完美交集
- ✍️ **爆款内容** — 50+标题公式+6大爆款类型+4周选题日历
- 📊 **数据复盘** — 4指标法持续优化你的内容策略
- 💰 **变现路线** — 从0粉到10万粉的完整变现指南
- 🔄 **5平台适配** — 一份内容适配小红书/抖音/B站/公众号/视频号
