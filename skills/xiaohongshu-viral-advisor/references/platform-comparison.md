## 跨平台内容适配策略

### 一鱼多吃原则
```
核心内容（1篇深度图文/视频）
    ↓
小红书 → 图文笔记（封面+正文+标签）
抖音 → 15-60秒竖版短视频
B站 → 3-10分钟横版解说视频
公众号 → 1500-3000字深度长文
视频号 → 30秒-1分钟竖版短视频+评论区导私域
```

### 适配优先级
1. **先发小红书** → 搜索流量长尾，生命周期最长
2. **再改编抖音/视频号** → 短视频二次创作
3. **最后发B站/公众号** → 深度内容延伸
