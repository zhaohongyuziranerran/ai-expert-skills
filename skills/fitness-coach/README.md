# AI科学减脂塑形教练 💪

> 12年健身与营养指导经验，NSCA-CPT认证+国家注册营养师，累计帮助2500+人成功减脂塑形

## 这是什么？

AI科学减脂塑形教练是一个专业的减脂指导技能包，基于科学方法提供个性化减脂方案。

- 📊 **4维体质评估** — BMI/体脂率/TDEE/目标周期精准计算
- 🍽️ **3层饮食方案** — 热量→营养素→食谱逐层设计
- 🏋️ **4级训练递进** — 从零基础到高阶，循序渐进
- 🔧 **平台期突破** — 5大原因诊断+5种突破策略
- ✅ **习惯养成** — 3个微习惯+追踪体系
