# TDEE详细计算公式与示例

## 一、基础代谢率（BMR）计算

### Mifflin-St Jeor公式（最推荐）
```
男性: BMR = 10 × 体重(kg) + 6.25 × 身高(cm) - 5 × 年龄 + 5
女性: BMR = 10 × 体重(kg) + 6.25 × 身高(cm) - 5 × 年龄 - 161
```

### 计算示例

**示例1：女性，28岁，165cm，65kg**
```
BMR = 10 × 65 + 6.25 × 165 - 5 × 28 - 161
    = 650 + 1031.25 - 140 - 161
    = 1380.25 kcal/天
```

**示例2：男性，35岁，175cm，80kg**
```
BMR = 10 × 80 + 6.25 × 175 - 5 × 35 + 5
    = 800 + 1093.75 - 175 + 5
    = 1723.75 kcal/天
```

## 二、活动系数

| 活动水平 | 描述 | 系数 |
|---------|------|------|
| 久坐 | 办公室工作，几乎不运动 | 1.2 |
| 轻运动 | 每周1-3次轻度运动 | 1.375 |
| 中运动 | 每周3-5次中等强度运动 | 1.55 |
| 重运动 | 每周6-7次高强度运动 | 1.725 |
| 极重运动 | 每天高强度训练或体力劳动 | 1.9 |

### 如何判断活动水平
- **久坐**：整天坐着，上下班开车，回家躺着
- **轻运动**：偶尔散步/瑜伽/慢跑，每周1-3次
- **中运动**：规律健身/跑步/游泳，每周3-5次
- **重运动**：每天训练1-2小时，或体力劳动工作
- **极重运动**：运动员/建筑工人/每天训练2小时+

## 三、TDEE计算

### 公式
```
TDEE = BMR × 活动系数
```

### 完整计算示例

**女性，28岁，165cm，65kg，轻运动**
```
BMR = 1380 kcal
TDEE = 1380 × 1.375 = 1897 kcal/天
```

**男性，35岁，175cm，80kg，中运动**
```
BMR = 1724 kcal
TDEE = 1724 × 1.55 = 2672 kcal/天
```

## 四、减脂热量目标

### 科学减脂原则
- **安全减重速度**：每周0.5-1kg
- **热量缺口**：TDEE - 300~500 kcal/天
- **最低热量底线**：女性≥1200kcal/天，男性≥1500kcal/天

### 热量目标计算
```
减脂目标热量 = TDEE - 500 kcal（推荐）
温和减脂 = TDEE - 300 kcal
快速减脂 = TDEE - 700 kcal（不建议长期）

每周减重预估：
- 300kcal缺口 → 约0.3kg/周
- 500kcal缺口 → 约0.5kg/周
- 700kcal缺口 → 约0.7kg/周
```

### 计算示例

| 案例 | TDEE | 温和减脂 | 推荐减脂 | 快速减脂 |
|------|------|---------|---------|---------|
| 女性28岁 65kg 轻运动 | 1897 | 1597 | 1397 | 1197(低于底线!) |
| 男性35岁 80kg 中运动 | 2672 | 2372 | 2172 | 1972 |

> ⚠️ 女性热量不要低于1200kcal/天！男性不要低于1500kcal/天！

## 五、体脂率估算

### 公式
```
男性体脂率 = 1.2 × BMI + 0.23 × 年龄 - 16.2
女性体脂率 = 1.2 × BMI + 0.23 × 年龄 - 5.4

BMI = 体重(kg) / 身高(m)²
```

### 体脂率标准

| 分类 | 男性 | 女性 |
|------|------|------|
| 必需脂肪 | 2-5% | 10-13% |
| 运动员 | 6-13% | 14-20% |
| 健康范围 | 14-17% | 21-24% |
| 可接受 | 18-24% | 25-31% |
| 肥胖 | 25%+ | 32%+ |

### 减脂目标体脂率建议
- 男性：目标12-18%（腹肌可见需<15%）
- 女性：目标20-25%（马甲线可见需<22%）

## 六、常见误区

### ❌ 误区1："吃的越少减的越快"
**真相**：热量缺口>700kcal会导致肌肉流失+代谢下降，反而更难减

### ❌ 误区2："基础代谢很高，少吃没事"
**真相**：长期低热量饮食会让BMR下降20-30%，恢复饮食后反弹更快

### ❌ 误区3："运动了就可以多吃"
**真相**：跑步30分钟消耗约200-300kcal，一杯奶茶就回来了

### ❌ 误区4："减脂=减体重"
**真相**：体重下降可能是水分+肌肉，体脂率才是关键指标

### ✅ 正确做法
1. 先算TDEE，再设300-500kcal缺口
2. 保证蛋白质1.6-2.0g/kg体重
3. 每周称重1次，2周不变才调整
4. 力量训练保肌肉，有氧训练消耗脂肪
