# 减脂平台期诊断流程图与突破策略

## 平台期判定标准

### 不是平台期的情况
- 体重3-5天没变 → 正常波动（水分/排便/饮食差异）
- 体重降了但体脂没变 → 可能在增肌减脂同时发生（好事！）
- 体重1周没变 → 可能是经期水分滞留（女性）

### 确认平台期的标准
- ✅ 体重连续**14天**无变化（在执行饮食+训练的前提下）
- ✅ 体脂率连续**14天**无变化
- ✅ 腰围/腿围连续**14天**无变化
- ✅ 确认饮食和训练都严格执行

## 诊断流程图

```
开始：体重2周不变
    ↓
Q1: 是否精确记录了所有饮食（包括零食/饮料/调味料）？
    → 否 → 开始记录3天，99%是隐性热量问题
    → 是 ↓

Q2: 每日实际摄入是否超过目标热量？
    → 是 → 回到饮食计划，严格记录
    → 否 ↓

Q3: 是否每周至少运动3次，强度达标？
    → 否 → 增加运动频率/强度
    → 是 ↓

Q4: 睡眠是否充足（7-8小时）？压力是否过大？
    → 否 → 皮质醇升高会导致水分滞留+食欲增加
    → 是 ↓

Q5: 体重降低后是否重新计算了TDEE？
    → 否 → 体重每降5kg需重算TDEE（会变低）
    → 是 ↓

Q6: 训练内容是否超过4周没变？
    → 是 → 身体已适应，需要新刺激
    → 否 ↓

→ 确认进入真正的平台期，执行突破策略
```

## 5大突破策略

### 策略1：饮食重置法（推荐首选）
```
执行周期：2周

Week 1: 恢复维持热量（TDEE水平）吃7天
  → 目的：恢复代谢激素水平，降低皮质醇
  → 体重可能反弹1-2kg（水分，不要慌）

Week 2: 重新开始减脂热量（TDEE-500）
  → 重新计算TDEE（用当前体重！）
  → 大概率会打破平台期
```

### 策略2：碳水循环法
```
执行周期：持续使用

高碳水日（训练日）：TDEE-200kcal，碳水3g/kg
低碳水日（休息日）：TDEE-600kcal，碳水1g/kg
循环：高-低-低-高-低-低-高

原理：高碳水日补充糖原+维持代谢
      低碳水日创造更大热量缺口
```

### 策略3：运动升级法
```
3种升级方式（选1种）：

A. 增加有氧量：每次训练+15分钟中等有氧
B. 替换运动方式：跑步→游泳，器械→自由重量
C. 引入HIIT：每周2次20分钟高强度间歇

注意：不要同时增加太多，每次只改1个变量
```

### 策略4：间歇性断食
```
16:8断食法：
- 进食窗口：12:00-20:00（8小时）
- 断食窗口：20:00-次日12:00（16小时）
- 进食窗口内正常吃减脂热量
- 断食窗口只喝水/黑咖啡/茶

原理：延长空腹时间→胰岛素水平更低→脂肪氧化更多
注意：不适合低血糖/孕妇/进食障碍者
```

### 策略5：减压恢复法
```
执行周期：1周

- 训练量减半（只做2-3次轻度运动）
- 热量恢复到TDEE水平
- 每天7-9小时睡眠
- 每天10分钟冥想/深呼吸
- 泡澡/按摩放松

适用人群：长期减脂（>3个月）+ 睡眠差 + 压力大
```

## 不同阶段平台期推荐策略

| 减脂阶段 | 平台期特征 | 推荐策略 |
|---------|-----------|---------|
| 1-2个月 | 体重突然不变 | 策略3（运动升级） |
| 2-4个月 | 体重缓慢下降后停滞 | 策略1（饮食重置）+ 策略3 |
| 4-6个月 | 体重完全不变+疲劳 | 策略5（减压恢复）→ 策略1 |
| 6个月+ | 反复平台+反弹 | 策略2（碳水循环）+ 策略4 |

## 心理调适

### 平台期最常犯的错误
1. ❌ 大幅减少热量（<1000kcal）→ 代谢崩溃
2. ❌ 大幅增加运动（每天2小时+）→ 过度训练
3. ❌ 放弃计划，开始暴食 → 前功尽弃
4. ❌ 频繁称重，焦虑加重 → 皮质醇升高

### 正确心态
1. ✅ 平台期是正常现象，不是失败
2. ✅ 体重不变≠脂肪不变，可能正在重组
3. ✅ 看围度比看体重更有意义
4. ✅ 给身体2-4周适应，不要急于调整
5. ✅ 每次只调整1个变量，2周后看效果
